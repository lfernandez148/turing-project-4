Welcome to Your Static Web Server

Your Flask static file server is running successfully with image protection!

Features:

  • ✅ Public static files served from web/www/
  • 🔒 Protected images require authentication
  • 🔑 Token-based authentication system
  • 📁 Organized folder structure

Image Structure:

web/www/images/
├── public/          → Publicly accessible
│   └── sample-logo.png
└── protected/       → Requires authentication
    └── confidential-data.png

API Endpoints:

Public Images: /images/public/filename.jpg
Protected Images: /images/protected/filename.jpg (Requires Authorization header)
Verify Token: /auth/verify

Example Usage:

Public Image (No Auth): 
curl http://localhost:8080/images/public/sample-logo.png
Protected Image (With Auth): 
curl -H "Authorization: Bearer your-secret-token" \
     http://localhost:8080/images/protected/confidential-data.png
Verify Token: 
curl -H "Authorization: Bearer your-secret-token" \
     http://localhost:8080/auth/verify
